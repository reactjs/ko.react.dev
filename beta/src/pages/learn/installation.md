## React 사용해보기 {/*try-react*/}

React를 사용하기 위해서 따로 설치할 것은 없습니다. sandbox를 사용해 보세요.

<Sandpack>

```js
function Greeting({ name }) {
  return <h1>Hello, {name}</h1>;
}

export default function App() {
  return <Greeting name="world" />
}
```

</Sandpack>
