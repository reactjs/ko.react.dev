<Intro>

React 개발은 전 세계 여러 회사와 커뮤니티의 기여자들이 이끌어 나갑니다.

</Intro>

React 작업은 여러 워킹 그룹으로 나뉘어 진행되며, 각 그룹은 Server, DOM, Fiber, Docs & Community, Compiler, DevX, React Native와 같이 프로젝트의 특정 영역을 담당합니다. 각 워킹 그룹은 React 리더십 위원회(Leadership Council)에 대표를 두며, 이 위원회가 프로젝트 전반의 방향을 조율합니다.
