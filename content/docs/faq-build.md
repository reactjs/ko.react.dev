### JSX에서 어떻게 주석을 달 수 있나요? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* 주석은 여기에 */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* 여러 줄의 
  주석도 가능합니다. */}
  Hello, {name}! 
</div>
```
